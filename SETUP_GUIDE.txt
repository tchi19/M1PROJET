## ðŸ”§ If Something Goes Wrong

### Database Not Initializing
```bash
# Check MySQL is running
# Manually create database:
mysql -u root -p
CREATE DATABASE exam_timetable;
```

### Can't Login
- Use same credentials you registered with
- Check email/password carefully (case-sensitive)
- Try creating a new account

### Blank Pages
- Check PHP is enabled on your server
- Verify all files uploaded correctly
- Check browser console for errors (F12)

### Missing Data in Tables
- Admin needs to create departments, formations, modules first
- Students need to be enrolled in modules
- Professors need to be assigned to modules before scheduling
